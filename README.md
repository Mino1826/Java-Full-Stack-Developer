# Employee Management System

Employee Management System is a full-stack web application for managing employee records.

The project includes a Spring Boot backend REST API and a React frontend application.

## Tech Stack

### Backend
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven

### Frontend
- React
- Vite
- Tailwind CSS
- JavaScript

### Tools
- Git
- GitHub
- Postman
- IntelliJ IDEA
- MySQL Workbench

## Features

- Create employee
- View employee list
- Update employee
- Delete employee
- REST API integration
- MySQL database connection
- Responsive frontend UI
